DocuX Privacy Policy

Last updated: June 2026

App: DocuX  ·  Developer: DJ Mirontos

1. Introduction

DocuX is a document scanner and organizer app that helps users store, organize, and search their screenshots and documents using on-device OCR and AI classification.

2. Information We Collect

  • Photos/screenshots you upload (stored locally on your device)
  • OCR text extracted from your images (stored locally)
  • AI-generated metadata: titles, tags, categories, summaries (stored locally)
  • Google account email (only if you sign in for cloud backup)
  • Cloud backup data stored securely in Supabase (only if you opt in)

3. How We Use Your Information

  • To provide on-device OCR text extraction via ML Kit (no data leaves your phone for this step)
  • To provide AI document classification via DeepSeek or Anthropic APIs (only extracted OCR text is sent, never the actual image)
  • To sync your data to cloud backup if you choose to enable it
  • We never sell your data to third parties

4. Third Party Services

  • Google Sign-In (OAuth 2.0) — for optional cloud backup authentication
  • Supabase — cloud database for optional backup storage
  • DeepSeek API — AI text classification (OCR text only)
  • Anthropic API — backup AI classification (OCR text only)
  • Google ML Kit — on-device OCR (no data transmitted)

5. Data Storage

All photos and data are stored locally on your device by default. Cloud backup is completely optional and requires Google Sign-In. You can delete all your data anytime from the app Settings screen.

6. Data Security

Local data is protected by your device's built-in security. Cloud backup data is encrypted at rest. DocuX also offers an optional App Lock feature (biometric or PIN).

7. Children's Privacy

DocuX is not intended for use by children under 13 years of age.

8. Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected on this page with an updated revision date.
